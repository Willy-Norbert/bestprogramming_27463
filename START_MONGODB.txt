========================================
HOW TO START MONGODB
========================================

MongoDB is installed but the service is stopped.
You need to start it to connect to the database.

METHOD 1: Services Window (Easiest - No Admin Needed)
------------------------------------------------------
1. Press Win + R
2. Type: services.msc
3. Press Enter
4. Find "MongoDB Server (MongoDB)" in the list
5. Right-click on it
6. Click "Start"
7. Wait a few seconds
8. Restart your backend server

METHOD 2: PowerShell as Administrator
--------------------------------------
1. Right-click Start Menu
2. Click "Windows PowerShell (Admin)" or "Terminal (Admin)"
3. Run: Start-Service -Name "MongoDB"
4. Or run: net start MongoDB
5. Restart your backend server

METHOD 3: Command Prompt as Administrator
------------------------------------------
1. Right-click Start Menu
2. Click "Command Prompt (Admin)"
3. Run: net start MongoDB
4. Restart your backend server

METHOD 4: Use MongoDB Atlas (Cloud - Recommended)
---------------------------------------------------
If you don't want to deal with local MongoDB:

1. Run: .\setup-mongodb-atlas.ps1
2. Follow the prompts
3. It will configure everything automatically
4. Restart your backend server

========================================

========================================

2. You should see: "âœ… Connected to MongoDB"
